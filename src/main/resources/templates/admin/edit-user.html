Редактировать Пользователя

Имя пользователя должно быть от 3 до 20 символов и содержать только буквы, цифры, точки, подчёркивания и дефисы.
Пароль должен содержать не менее 6 символов.
Пожалуйста, укажите имя. Максимальная длина — 50 символов.
Пожалуйста, укажите фамилию. Максимальная длина — 50 символов.
Пожалуйста, укажите корректный email.
Пожалуйста, укажите телефон в формате: +7 (XXX) XXX-XX-XX.
Пожалуйста, выберите хотя бы одну роль.
